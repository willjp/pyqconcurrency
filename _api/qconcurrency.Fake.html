class qconcurrency.Fake(*args, **kwds)[source]¶

Bases: object

A fake object, that will allow you to call attributes (at any level) as functions with any number of args/kwds without errors.

You can use this in place of a signalmgr (see SignalManagerFactory) if you want the ability to run methods inside or outside of threads.
